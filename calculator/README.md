## Simple Calculator

### Description

A simple calculator which takes input as two values and allows the following operation

1. Addition
2. Subtraction
3. Multiplication
4. Division

### Prerequisites

- [Golang](https://go.dev/)

### Installation

#### Clone the respository

```shell
git clone https://github.com/mrarslanark/golang.git
```

#### Run the application

```shell
cd calculator && go run .
```
